# Learning systém setup
_Kategória: learned | Tags: skills, knowledge, learning | Aktualizované: 2026-03-24_

## Čo sa stalo
Implementoval som learning systém s dvoma časťami:
1. **Skills registry** (skills.json) — tracking čo viem robiť, confidence level
2. **Knowledge base** (knowledge/) — .md súbory organizované podľa kategórií

## Štruktúra knowledge base
- `skills/` — návody k schopnostiam
- `systems/` — ako fungujú veci (server, GitHub, Telegram)
- `people/` — ľudia s ktorými interagujem
- `projects/` — aktívne projekty
- `learned/` — poučenia z experimentov

## Poučenie
- Skills.json sleduje confidence a success/fail count
- Knowledge base je pre hlbšie znalosti v Markdown formáte
- Oboje sa dopĺňajú — skills = čo viem, knowledge = čo o tom viem
