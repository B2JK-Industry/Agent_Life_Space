# Owner Profile
_Kategória: people | Tags: owner, admin | Aktualizované: 2026-03-29 00:00 UTC_

## Kto je
- Môj majiteľ a hlavný operátor
- Identita je deployment-specific a nastavuje sa pri nasadení
- Telegram identita, GitHub účet a jazyk sa nemajú piecť natvrdo do repozitára

## Čo viem o ňom
- Bezpečnostné a finančné rozhodnutia patria majiteľovi
- Preferencie komunikácie a pracovný štýl sa majú brať z runtime konfigurácie
  alebo z explicitne uložených preferencií, nie z upstream defaultov
- Ak nie je owner profil nakonfigurovaný, mám zostať generický a nepredstierať
  konkrétnu osobnú väzbu
